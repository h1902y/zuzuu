# pi as a Host (capture + gate) — Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans. Steps use checkbox (`- [ ]`) syntax.

**Goal:** `mns enable --host pi` installs a pi extension that fires live capture + gates tool calls (block on `deny`); a new **pi adapter** (the 5th) parses pi's session JSONL. Brings pi to capture+gate parity as a wrapped host.

**Architecture:** Design B. The pi extension (`.pi/extensions/mns.ts`, `.ts` loads natively) fires `mns hook <event> --host pi --session <sessionFilePath>` (capture, detached) and, on `tool_call`, spawns the shared mns gate (`spawnSync`, awaited) returning `{block:true,reason}` on `deny`. Real-wire-confirmed (`tests/fixtures/hooks/pi.probe.jsonl`): events fire headless; `tool_call={toolName,input}`; session path via `ctx.sessionManager.getSessionFile()`.

**Tech Stack:** Node ≥22, ESM, zero deps, `node:test`. pi runtime executes the `.ts` extension (jiti); the extension spawns `node` for capture/gate.

## Real-wire facts
- **Session JSONL** (`~/.pi/agent/sessions/--<cwd-dashes>--/<ts>_<uuid>.jsonl`): line 1 = `{type:"session", version, id, timestamp, cwd}`; entries = `{type, id, parentId, timestamp, ...}` — `model_change`, `thinking_level_change`, `message {message:{role, content:[{type:"text",text}|{type:"toolCall",id,name,arguments}], usage}}`, and tool-result entries. Tree by `id`/`parentId` (8-char hex). Use the OpenRouter probe session (`~/.pi/agent/sessions/--Users-hkc-Documents-mns-pi-probe--/*.jsonl`, which has a real `bash` tool call) for goldens.
- **Events** (golden `tests/fixtures/hooks/pi.probe.jsonl`): `session_start{reason}` · `turn_start/turn_end` · `tool_call{toolName,toolCallId,input}` (gate) · `tool_result` · `session_shutdown`. No session id in event payloads → path via `ctx.sessionManager.getSessionFile()`.

- [ ] **Step 1: Add the pi branch + extension template** (`mns/commands/enable.mjs`)

Add a path helper + template near the opencode one:
```javascript
const piExtPath = (cwd) => join(repoRoot(cwd), '.pi', 'extensions', 'mns.ts');
const piExtension = () => `// installed by \`mns enable --host pi\` — live capture + guardrails gate (graceful).
import { spawn, spawnSync } from "node:child_process";
const NODE = ${JSON.stringify(NODE)};
const MNS = ${JSON.stringify(BIN)};
export default function (pi) {
  const fire = (event, ctx) => {
    try {
      const file = ctx?.sessionManager?.getSessionFile?.();
      if (!file) return;
      spawn(NODE, [MNS, "hook", event, "--host", "pi", "--session", file], { stdio: "ignore", detached: true }).unref();
    } catch {}
  };
  pi.on("session_start", async (_e, ctx) => fire("session_start", ctx));
  pi.on("turn_end", async (_e, ctx) => fire("turn_end", ctx));
  pi.on("session_shutdown", async (_e, ctx) => fire("session_shutdown", ctx));
  // gate: tool_call → run the shared mns gate; block on deny. Fail-open.
  pi.on("tool_call", async (event, ctx) => {
    try {
      const file = ctx?.sessionManager?.getSessionFile?.();
      const payload = JSON.stringify({ tool_name: event?.toolName, tool_input: event?.input, session_id: file });
      const res = spawnSync(NODE, [MNS, "hook", "PreToolUse", "--host", "pi"], { input: payload, encoding: "utf8", timeout: 5000 });
      const out = (res && res.stdout) || "";
      let decision = null;
      for (const line of out.split("\\n")) { const t = line.trim(); if (t.startsWith("{")) { try { decision = JSON.parse(t); } catch {} } }
      if (decision && decision.decision === "deny") return { block: true, reason: decision.reason || "blocked by mns guardrail" };
    } catch {}
    return undefined; // allow / no-match / any error → proceed (fail-open)
  });
}
`;
```
